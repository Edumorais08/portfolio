Projetos

  • Gestão Acadêmica

    Este é o projeto final da disciplina "Orientação a objetos". Desenvolvido em Java, este sistema permite o cadastro e gerenciamento de alunos, professores, disciplinas e turmas.

  • ChamaControl

    O ChamaControl é uma plataforma intuitiva, onde cidadãos possam ter acesso e monitorar informações sobre queimadas que possam estar acontecendo pelo território brasileiro. Utiliza React JS para o front-end, Node para o scraper dos dados e para o back-end e Sql para o banco de dados.

  • Filmes nos cinemas

    Esse projeto se trata de um site feito com React JS que utiliza a API "TMDB" para fazer a listagem de filmes que estão nos cinemas ou vão entrar em breve, tendo a possibilidade de escolher os filmes favoritos e fazer login na página.

  • Emmy Winners

    Uma aplicação com flask que faz a busca de todos os indicados ao prêmio emmy awards para séries de TV com webscraping utilizando python, salva em arquivo json e depois faz uma pagina para cada ano e categoria e lista todos eles.
